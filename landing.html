Resolvemos tus
dudas

GeoTopo usa datos 100% abiertos: elevaciones de NASA SRTM y ASTER (30m globales), IGN France (1m en Francia), USGS 3DEP (~10m en USA). Los datos de edificios, calles y capas urbanas provienen de OpenStreetMap, el mayor proyecto cartográfico colaborativo del mundo.
Sí. Los datos de OpenStreetMap tienen licencia ODbL que permite uso comercial con atribución. Los datos de NASA/SRTM son dominio público. GeoTopo no añade restricciones adicionales sobre los datos exportados.
Depende del dataset de elevaciones. SRTM y ASTER tienen precisión vertical de ±10m y resolución horizontal de 30m — suficiente para estudios preliminares y contexto. Para proyectos que requieren mayor precisión, recomendamos complementar con datos LiDAR locales.
GeoTopo requiere internet para descargar las elevaciones y los datos OSM. Una vez descargados, el procesamiento (curvas, triangulación, exportación) ocurre completamente en tu navegador — nada se envía a un servidor. Para uso sin internet, recomendamos descargar el archivo y usar Python: cd carpeta && python -m http.server 8080.
Puedes cancelar en cualquier momento desde tu panel de cuenta. No hay permanencia mínima. Si cancelas, mantienes el acceso Pro hasta el final del período ya pagado.
